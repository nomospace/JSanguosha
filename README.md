# 🦞 三国杀 Mini

> 纯 JavaScript 实现的三国杀单机版 | 32 武将 | 108 卡牌 | AI 自动对战

## 🌐 访问地址

- **本地**: http://localhost:3005/
- **公网**: http://47.102.199.24:3005/
- **GitHub**: https://github.com/nomospace/JSanguosha

## ✨ 特性

### 🎮 核心功能
- ✅ **32 位标准包武将** - 魏蜀吴群四大势力各 8 位
- ✅ **108 张标准包卡牌** - 基本牌、锦囊牌、装备牌完整实现
- ✅ **AI 自动对战** - 智能出牌策略
- ✅ **完整游戏规则** - 距离计算、装备系统、技能触发
- ✅ **响应式 UI** - 支持移动端和桌面端
- ✅ **新手引导** - 5 步流程快速上手

### 🎨 界面设计
| 功能 | 描述 |
|------|------|
| 势力色标识 | 魏（蓝）、蜀（绿）、吴（红）、群（橙） |
| 体力显示 | 「3❤」数字 + 心形格式 |
| 濒死效果 | 红光闪烁动画 |
| 阶段提示 | 摸牌/出牌/弃牌阶段标签 |
| 卡牌弹窗 | 点击查看卡牌详情 |
| 快捷操作 | 一键出杀/出桃/弃牌/结束回合 |

## 🎯 游戏流程

1. **开始游戏** - 随机选择 4 位武将
2. **初始发牌** - 每位玩家 4 张初始手牌
3. **回合制对战** - AI 自动出牌
4. **技能触发** - 武将技能自动触发
5. **游戏结束** - 最后存活者获胜

### 回合流程
```
准备阶段 → 摸牌阶段 → 出牌阶段 → 弃牌阶段 → 结束阶段
```
