## 🎯 操作步骤

### 1️⃣ 打开游戏
浏览器访问上述地址，看到加载页面：
```
🦞 微杀杀 Online
正在加载游戏资源...
```

### 2️⃣ 主菜单
加载完成后看到主菜单：
```
┌─────────────────────────────────┐
│     微杀杀 Online                │
│   视觉化三国杀单机版             │
│                                 │
│   [🎲 开始游戏]                 │
│                                 │
│   [👥 4 人局] [👥 6 人局]        │
│         [👥 8 人局]              │
└─────────────────────────────────┘
```

### 3️⃣ 选择人数
- **4 人局** - 快速对战（推荐）
- **6 人局** - 标准局
- **8 人局** - 身份局

### 4️⃣ 开始游戏
点击 **"🎲 开始游戏"** 按钮

### 5️⃣ 观看对战
游戏自动进行：
- 🃏 自动摸牌（2 张/回合）
- 🎯 AI 自动出牌
- ❤️ 自动使用桃回复
- 💀 阵亡玩家自动退出

### 6️⃣ 下一回合
点击 **"⏭️ 下一回合"** 继续

### 7️⃣ 游戏结束
最后一人获胜，显示：
```
🏆 胜利！
```

## 🎬 动画效果

### 摸牌动画
卡牌从右上角牌堆 → 飞到手牌区

### 出牌动画
卡牌从当前玩家 → 飞向目标玩家

### 伤害效果
```
💔 -1  ← 红色数字弹出 + 粒子爆发
```

### 回复效果
```
❤️ +1  ← 绿色数字弹出 + 粒子效果
```

### 阵亡效果
- 灰色粒子爆发
- 屏幕震动
- 玩家卡片变灰（透明度 50%）

### 胜利效果
- 彩带粒子从天而降
- 🏆 胜利文字动画
- 缩放脉冲效果

## 🐛 常见问题

### Q: 游戏加载慢？
A: 首次加载需要下载 1.2MB 资源，耐心等待进度条完成。

### Q: 画面卡顿？
A: 检查浏览器性能，关闭其他标签页。

### Q: 声音？
A: 当前版本暂无音效，预留接口待添加。

### Q: 可以手动出牌吗？
A: 当前为纯 AI 对战模式，手动模式开发中。
